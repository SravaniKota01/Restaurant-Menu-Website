# 🍽️ Chef Sravani's Kitchen

Welcome to **Chef Sravani's Kitchen**, a simple and attractive restaurant menu website built using **HTML5** and **CSS3**.

This project showcases a modern restaurant menu with neatly organized food categories, prices, restaurant timings, and a thank-you message. It is designed as a beginner-friendly project to practice front-end web development concepts.

## 📌 Features

- 🍴 Attractive restaurant menu layout
- 🥗 Separate food categories
  - Starters
  - Main Course
  - Fast Food
  - Beverages
  - Desserts
- 💰 Food items with prices
- 🎨 Beautiful background image
- 📱 Clean and responsive design
- 😊 Customer thank-you message
- 👩‍🍳 Professional menu card using CSS

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📚 Concepts Practiced
